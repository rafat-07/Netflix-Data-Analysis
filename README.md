# Netflix Data Analysis

This project analyzes Netflix movies and TV shows data to uncover insights about genre distribution, content trends, and platform growth using Python.
## About Me

I am an MSc Data Science & Analytics student at Teesside University London with a background in mathematics. I am interested in data analysis and machine learning, and I enjoy extracting insights from real-world datasets.
## Dataset

The dataset used in this project is the Netflix Movies and TV Shows dataset from Kaggle.

It contains information such as:
- Title
- Type (Movie or TV Show)
- Cast
- Country
- Release Year
- Genre (listed_in)
## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
## Project Workflow

1. Data loading and inspection  
2. Data cleaning and preprocessing  
3. Genre analysis using splitting and explode  
4. Visualization of top genres  
5. Analysis of Movies vs TV Shows  
6. Trend analysis of content over time
## Key Insights
- Drama is the most common genre in the dataset, showing that it represents a significant portion of Netflix’s overall content catalogue.
- Netflix content grew gradually until around 2015, followed by a sharp increase in the number of titles.
- The apparent decline after 2020 is likely due to incomplete data for recent years rather than an actual decrease..
- Movies are more prevalent than TV Shows on the platform.
## Visualizations

The project includes:
- Bar chart of top 10 genres
- Pie chart of Movies vs TV Shows
- Line chart showing content growth over time
